# Apache TinkerPop (apache-tinkerpop)
Apache TinkerPop is a graph computing framework for both graph databases (OLTP) and graph analytic systems (OLAP). It provides the Gremlin graph traversal language which works with any TinkerPop-enabled graph system.

## APIs

### Apache TinkerPop Gremlin API
TinkerPop provides the Gremlin graph traversal language and API, Gremlin Server with REST and WebSocket endpoints, and client drivers in Java, Python (gremlinpython), JavaScript, .NET, and Go for graph database interactions.
